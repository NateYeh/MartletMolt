# 🚜 MartletMolt - 後勤維護腳本 (Logistics Scripts)

本目錄存放用於系統維護、環境同步及數據修復的實用腳本。這些腳本主要服務於開發流程中的環境部署與運行的穩定性。

## 📋 腳本列表

- **`sync_systems.py`**: **核心維護腳本**。用於在 A/B 系統切換後，自動將變更同步至對應系統，確保環境一致性。
- **`fix_yaml.py`**: 自動偵測並修復無效或格式錯誤的 YAML 配置文件。
- **`validate_yaml.py`**: 驗證系統配置文件（如 `config.yaml`）是否符合預期的 Schema 架構。

## ⚠️ 使用須知
- 執行腳本前，請確保已激活專案虛擬環境。
- 涉及系統同步的腳本建議在 `orchestrator` 的監控下或手動確認後執行。

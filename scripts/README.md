# 🏗️ MartletMolt 基礎設施腳本 (Infrastructure Scripts)

本目錄存放用於維護系統基層運行的腳本。這些腳本通常由人類開發者手動執行，或由守護進程調用。

## 📜 腳本清單

| 檔案 | 功能描述 | 使用時機 |
| :--- | :--- | :--- |
| `start_services.sh` | 啟動 screen 守護進程 | 伺服器開機或手動重啟全套服務時。 |
| `sync_systems.py` | System A/B 代碼同步 | 完成任務開發，需要將變更從 A 同步至 B（或反之）時。 |

## ⚠️ 注意事項
- 這裡不應該存放「業務邏輯」或「AI 技能」。
- 若發現某個腳本是為了處理特定數據（如 YAML 修復、代碼檢查），請將其遷移至 `../../skills/` 目錄下封裝。
